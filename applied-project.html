Applied Learning Project

Project Overview

The Applied Learning Project is designed to provide you with hands-on experience, allowing you to synthesize and apply the knowledge and skills gained throughout the Blockchain Specialization. This capstone project will challenge you to build a functional blockchain-based solution.

Learners will apply the concepts covered in the courses to:

  • Deploy an instance of a blockchain.
  • Carry out peer-to-peer transactions on your deployed blockchain.
  • Program and thoroughly test smart contracts using a language like Solidity.
  • Analyze, design, and code a blockchain-based solution for a decentralized application (DApp).

This project will serve as a significant portfolio piece, demonstrating your practical abilities in blockchain development.

Key Project Phases [&] Deliverables (Example)

  1. Conceptualization [&] Design:
    • Define the problem your DApp will solve.
    • Design the DApp architecture, including smart contract interactions and user flows.
    • Document your design choices.
  2. Smart Contract Development:
    • Write, compile, and test your smart contracts.
    • Implement core logic and data structures.
  3. DApp Front-End Development:
    • Develop the user interface for interacting with your smart contracts.
    • Integrate wallet connections and transaction handling.
  4. Deployment [&] Testing:
    • Deploy your smart contracts to a test network (and optionally a private blockchain instance).
    • Conduct end-to-end testing of your DApp.
  5. Documentation [&] Presentation:
    • Prepare a final report detailing your project, design, implementation, and challenges.
    • (Optional) Prepare a short video demonstration.
